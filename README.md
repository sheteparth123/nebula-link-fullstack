# 🚀 NebulaLink — Secure URL Shortener

> A full-stack URL shortening service built with **Spring Boot, React, and PostgreSQL**  
> Focused on **security, scalability, and clean API design**

---

## 🌐 Overview

NebulaLink allows users to shorten long URLs and manage them securely through authentication-based access.

It is designed to simulate real-world backend systems with proper architecture, authentication, and deployment.

---

## ✨ Features

- 🔗 Shorten long URLs instantly  
- 🔐 JWT-based Authentication & Authorization  
- 📊 User-specific URL management  
- ⚡ Fast redirection system  
- 🌍 RESTful API architecture  
- 🐳 Dockerized for easy deployment  

---

## 🛠️ Tech Stack

### 🔹 Backend
- Java 17  
- Spring Boot  
- Spring Security (JWT)  
- JPA / Hibernate  

### 🔹 Frontend
- React (Vite)  
- Axios  
- Tailwind CSS  

### 🔹 Database
- PostgreSQL  

### 🔹 DevOps
- Docker  
- Render (Deployment)
